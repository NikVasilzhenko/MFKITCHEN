Белковые бобы привносят пикантный вкус в сегодняшние вегетарианские бургеры. Фасоль пюре с семенами чиа и сливочным веганским паприкой-дижонским айоли, чтобы сформировать сытные котлеты, которые обжариваются до золотисто-коричневого цвета. Они укладываются между булочками бриоши с хрустящим капустным салатом, усеянным сыром котиха, для сливочного укуса. Бонус: Этот рецепт делает четыре котлеты! Используйте дополнения в качестве начинки для обертываний или начинки для салатов.

Информация об аллергенах:
